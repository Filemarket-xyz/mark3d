# Бекенд индексер mark3d коллекций
## Установка зависимостей
Установка abigen'а:
```shell
go install github.com/ethereum/go-ethereum@v1.10.26
cd $GOPATH/pkg/mod/github.com/ethereum/go-ethereum@v1.10.26
make devtools
```
